# 使用可能なモバイル SDK
{: #sdk}

{{site.data.keyword.cloud}} Web サービスおよびモバイル・サービスの SDK をアプリに追加するには、使用する SDK を選択し、それらの SDK をアプリにプルするように依存関係マネージャーを構成します。

{: #server_sdk}
