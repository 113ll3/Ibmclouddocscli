# Plug-ins für die {{site.data.keyword.Bluemix_notm}}-Befehlszeilenschnittstelle (ibmcloud cli)
{: #cli_plugins}

Sie können Ihre {{site.data.keyword.Bluemix_notm}}-Befehlszeilenschnittstelle (ibmcloud cli) mit Plug-ins erweitern. Sie können Plug-ins, die in [CLI Plug-in Repository ![Symbol für externen Link](../icons/launch-glyph.svg)](http://plugins.ng.bluemix.net/){: new_window} verfügbar sind, installieren und verwenden. Jedes Plug-in wird durch seinen Binärdateinamen, seinen vom Entwickler definierten Plug-in-Namen und durch die Befehle, die das Plug-in bereitstellt, identifiziert. Zur Installation eines Plug-ins können Sie nur den Binärdateinamen verwenden. Für jede andere Aktion können Sie den Plug-in-Namen oder einen Befehl verwenden.
{:shortdesc}
