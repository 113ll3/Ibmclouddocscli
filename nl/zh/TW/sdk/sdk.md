# 可用的行動 SDK
{: #sdk}

若要將 {{site.data.keyword.cloud}} Web 及行動服務 SDK 新增至您的應用程式，請選取您要使用的 SDK，並配置相依關係管理程式以將 SDK 取回至應用程式。

## 伺服器 SDK
{: #server_sdk}

如果您有 Java、NodeJS 或 Swift 伺服器應用程式，可以使用下列 SDK 來與各自服務通訊。

### {{site.data.keyword.appid_short}} 伺服器 SDK
{: #appid_sdk}
