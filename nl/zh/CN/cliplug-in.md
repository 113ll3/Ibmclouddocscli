# Cloud Foundry 命令行界面插件

您可以安装并使用 [CLI 插件存储库](http://plugins.ng.bluemix.net/){: new_window}上提供的 Cloud Foundry 命令行界面 (cf cli) 插件。每个插件通过其二进制文件名、其开发者定义的插件名称以及该插件提供的命令进行识别。二进制文件名只能用于安装插件，而插件名称或命令可以用于执行其他任何操作。
{:shortdesc}
